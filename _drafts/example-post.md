## Important Note

**This file is in the `_drafts/` folder, which means it will NOT appear on the live site when deployed to GitHub Pages.**

To publish this post:
1. Move it to the `_posts/` folder
2. Rename it to include the date: `YYYY-MM-DD-example-post.md`
3. Update the `date` field in the front matter
4. Commit and push

You can preview drafts locally by running:
```bash
jekyll serve --drafts
```
